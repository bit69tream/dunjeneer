# dunjeneer

just a playground, nothing in particular.

# building & running
```bash
$ cmake -B build
$ cmake --build build

$ build/dunjeneer
```

# controls

| Key          | Action                             |
|--------------|------------------------------------|
| <kbd>E</kbd> | Move up                            |
| <kbd>S</kbd> | Move left                          |
| <kbd>D</kbd> | Move down                          |
| <kbd>F</kbd> | Move right                         |
| Left click   | Drill                              |
| Right click  | Do something with the focused tile |
| <kbd>X</kbd> | X-ray                              |
